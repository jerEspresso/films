# Images — The Grandmaster

Drop your own film stills in this folder, then update the `src` of each `<figure>`
in [`../the-grandmaster-scenes-explained.html`](../the-grandmaster-scenes-explained.html)
to point at the file.

Every image slot currently shows `placeholder.svg`. Replace each
`src="images/placeholder.svg"` with the matching file below.

| Slot (reading order) | Suggested filename | Scene |
|---|---|---|
| 1 | `01-opening-rain-fight.jpg` | The opening fight in the rain |
| 2 | `02-golden-pavilion.jpg` | The Golden Pavilion stairwell (Gong Er vs Ip Man) |
| 3 | `03-gong-er-vow.jpg` | 奉道 — Gong Er cuts her hair / takes the vow |
| 4 | `04-train-station-showdown.jpg` | The snowy Manchurian train-station showdown |

## Notes
- Recommended width ~1200px; JPG or WebP keeps the file size down.
- Filenames are only suggestions — any name works as long as the `src` matches.
- Use your own screenshots or properly licensed images: the film is copyrighted
  (Block 2 Pictures / Annapurna et al.).
