# Theme Preview

This page is written in **Markdown** and rendered through the Jekyll theme set in
`_config.yml` (currently `jekyll-theme-cayman`). It exists so you can see the theme
applied to real content — and to confirm the setup works.

Your film companion pages are complete, self-contained HTML with their own design,
so Jekyll serves them **as-is**: the theme does not override them. The theme styles
Jekyll-rendered pages like this one.

## What the theme styles

- Headings, paragraphs, and lists like this one
- [Links — e.g. back to the collection](./)
- Blockquotes and code blocks:

```bash
# Any new Markdown page with front matter gets the theme automatically.
echo "hello, themed world"
```

> **Note:** This file (`theme-preview.md`) is only a demonstration — delete it
> whenever you like and the rest of the site is unaffected.
